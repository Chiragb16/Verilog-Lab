# Overview

Verilog Lab is a personal practice repository containing digital logic designs implemented in Verilog HDL.

The repository documents my learning journey through combinational logic, sequential logic, arithmetic circuits, counters, memory elements, and simulation-based verification using ModelSim.

Each module was written from scratch to strengthen my understanding of RTL design and digital hardware concepts.
